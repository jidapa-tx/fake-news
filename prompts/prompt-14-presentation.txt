## Project Context (use this to inform every slide)

**App name:** ชัวร์ก่อนแชร์
**Tagline:** ตรวจสอบข่าวก่อนส่งต่อ — เพราะข่าวปลอมแพร่เร็วกว่าที่คุณคิด

**Stack:** Next.js 14 (App Router) + TypeScript strict mode, TailwindCSS + shadcn/ui, PostgreSQL 15 + Prisma ORM, Zustand + TanStack Query, Zod validation, Google Gemini API (gemini-2.5-flash), next-intl (TH/EN)

**Deployment:** Local Docker Compose only (MVP scope — no production deploy, no auth)

**Features (FR.1–FR.7):**
- FR.1 — No-auth access; history stored in localStorage only; privacy-first
- FR.2 — Multi-format input (text / URL / image); 5-level credibility score (0–100); evidence breakdown; AI confidence indicator; Thai reasoning bullets
- FR.3 — Reference list with stance badges; source credibility sorting; 20+ pre-seeded Thai and international trusted sources
- FR.4 — Domain analysis (age, SSL, WHOIS, phishing); social account analysis (MVP: mock data); coordinated behavior detection (MVP: seeded demo)
- FR.5 — AI-generated image detection via Gemini; image metadata/EXIF analysis; reverse image search (MVP: mock)
- FR.6 — Trending searches page (anonymized, hashed queries); period filter (day/week/month); clickable entries
- FR.7 — Local history page; search & filter; export/import JSON

**5 Verdict Levels:**
- อันตราย (0–20, red) / น่าสงสัย (21–40, orange) / ไม่แน่ใจ (41–60, yellow) / ค่อนข้างจริง (61–80, lime) / ยืนยันแล้ว (81–100, green)

**Key Technical Decisions:**
- All history in localStorage — no personal data on server
- Query hashing (SHA-256) for trending — raw text never stored
- Rate limiting: 30 req/min per IP (in-memory sliding window)
- Single external API: Google Gemini (image only); all other analysis uses DB lookups + scoring logic
- Graceful fallback if Gemini fails — endpoint always returns 200 with fallback data
- Scoring logic: base 50 → ±8 per trusted source stance → −25 suspicious domain → −20 partial fake claim match
- Zod validation on both client and server
- API errors always: `{ error: { code, message_th, message_en } }` — never raw stack traces
- i18n: Thai default, English supported via next-intl

**Architecture:**
```
Docker Compose
├── app (Next.js, port 3000)
│   ├── /api/analyze/text    POST — text & URL analysis
│   ├── /api/analyze/image   POST — Gemini AI image analysis
│   ├── /api/analyze/source  POST — domain risk analysis
│   ├── /api/trending        GET  — trending queries
│   └── /api/references/:id  GET  — reference list
└── db (PostgreSQL 15, port 5432)
    └── Tables: Analysis, Reference, TrustedSource,
               KnownFakeClaim, SuspiciousDomain, TrendingEntry

External: Google Gemini API (image analysis only)
```

**Screens:** Home (`/`) / Result (`/result/[id]`) / History (`/history`) / Trending (`/trending`)
